## Users API

**GET: Get All Users**
```
/api/users
```

**POST: Create New User**
```
/api/users
```

**PUT: Update User**
```
/api/users/:userId
```

**DELETE: Delete User**
```
/api/users/:userId
```

## Collection API

**GET: Get All Collections**
```
/api/collection
```

## Shop API

**GET: Get Shop Page**
```
/api/shop/:contractAddress/:page
```

## Product API

**GET: Get One Product Details**
```
/api/product/:contractAddress/:tokenId
```

**GET: Get One Transaction**
```
/api/product/tx/:contractAddress/:tokenId
```
